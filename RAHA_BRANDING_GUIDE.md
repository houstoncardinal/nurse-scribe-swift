## üìù Typography

### Primary Font Family: Inter
**Weights Used:**
- Light (300) - Subtle labels
- Regular (400) - Body text
- Medium (500) - Secondary headings
- SemiBold (600) - Primary headings
- Bold (700) - Strong emphasis

### Heading Hierarchy
- **H1**: 2xl (24px) - Page titles
- **H2**: xl (20px) - Section headers
- **H3**: lg (18px) - Card headers
- **H4**: base (16px) - Component headers

### Body Text
- **Large**: base (16px) - Primary content
- **Regular**: sm (14px) - Secondary content
- **Small**: xs (12px) - Metadata, labels
